# PriconeTL
 Princess Connect! ReDive Translation Project
# About it
 這裡是為了將日文翻譯成繁體中文而設置，請將`AutoTranslatorConfig.ini`檔案內
 ```
 [General]
 Language=en
 FromLanguage=ja
 ```
 更改為
 ```
 [General]
 Language=zh-TW
 FromLanguage=ja
 ```
 以套用繁體中文設置。
